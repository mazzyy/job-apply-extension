Settings

LLM provider

Pick where the AI work runs. Cloud uses Azure OpenAI (gpt-5-mini) — fastest and highest quality, costs ~$0.50/day at moderate use. Local uses Ollama on this machine — free and private, slower, slightly weaker on deep reasoning. Hybrid is the recommended default — routine tasks go local, deep-reasoning tasks go cloud.

Per-task overrides (hybrid mode only)
Pick provider for each kind of LLM call. Empty = use the hybrid default.

Azure OpenAI credentials

Used for Cloud and Hybrid modes. Paste your key here so you don't have to edit config.py or .env. Stored in your local database — never sent anywhere except Azure itself.

Network

The extension and dashboard talk to a FastAPI backend running on your machine. Default: http://localhost:8000.
